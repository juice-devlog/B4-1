# My Portfolio

## 사용 기술
- HTML5, CSS3, Vanilla JavaScript (ES6+)
- GitHub Pages (배포)
- GitHub REST API

## 주요 기능
- 반응형 레이아웃 (모바일/태블릿/데스크톱)
- 다크모드 (localStorage 유지)
- GitHub API 연동 프로젝트 카드
- 폼 유효성 검사
- 스크롤 애니메이션 (Intersection Observer, threshold: 0.2)
- 스크롤 탑 버튼 (300px 기준)
- 헤더 배경 변경 (60px 기준)
